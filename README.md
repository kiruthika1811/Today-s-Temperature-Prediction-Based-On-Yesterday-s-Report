# 🌡️ Temperature Prediction Based on Yesterday's Temperature

## 📌 Project Overview
This project predicts the next day's temperature using the previous day's temperature as input. It is a simple Machine Learning application that demonstrates how historical weather data can be used to forecast future temperatures. The model is trained using a dataset containing daily temperature records and generates predictions with good accuracy.

## 🎯 Features
- Predicts the next day's temperature.
- Uses historical weather data.
- Data preprocessing and cleaning.
- Machine Learning model training.
- Model performance evaluation.
- Graphical visualization of actual vs predicted temperatures.
- Simple and beginner-friendly implementation.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook / VS Code

## 📂 Dataset
The dataset contains daily temperature records downloaded from Kaggle. The previous day's temperature is used as the input feature to predict the current day's temperature.

## ⚙️ Workflow
1. Load the dataset.
2. Preprocess the data.
3. Select yesterday's temperature as the input feature.
4. Split the dataset into training and testing sets.
5. Train the Machine Learning model.
6. Predict temperatures on the test data.
7. Evaluate model performance.
8. Visualize the results.

## 🤖 Machine Learning Algorithm
- Linear Regression

## 📊 Output
The model predicts the next day's temperature and compares it with the actual temperature. The results are displayed in both numerical and graphical formats.
